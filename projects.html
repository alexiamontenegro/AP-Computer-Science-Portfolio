My App Project Will be shown here!

The purpose of my this project was to create an app that would help people's lives. With this project I worked with a partner and created an Activity Generator. The Activity Generator was designed to for people who are bored and don't know what to do, find something to do! The app begins by giving a choice of whether the activity the user wants to do is by themselves, with friends, in the morning, at night, indoor, or outdoor. After the user decides, the Activity Generator will generate an activity. If the user does not like the activity they can regenerate and another activity will show up. If they do like the activity that was given they can select "will do" that will transfer the activity to a list of all the activities they have done. The users can view the list at a later time. While making this app, I was able to learn a new way of coding and how to debug my program.
